
### 1. Entre na pasta backend e instale as dependências
```
npm install
```
### 2. Faça a migration do banco de dados
```
npx prisma db push
```
### 3. Rode o backend
```
npm run dev
```
### 4. Entre na pasta frontend e instale as dependências
```
npm install
```
### 5. Execute o frontend
```
npm start
```
